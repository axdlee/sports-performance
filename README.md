# 体育成绩评估系统

## 项目简介
适用于中学/高校体育成绩评估的桌面端客户端程序，支持macOS与Windows平台。通过用户输入个人体育成绩数据，系统根据国家学生体质健康标准自动计算得分、统计总分、生成成绩分析报告与发展趋势曲线，并提示弱项。

## 功能特点
- 支持男女生不同评分标准
- 必选项：男生1000米跑/女生800米跑（10分）
- 选考项目：第一类（10分）+ 第二类（10分）
- 历史成绩记录与趋势分析
- 弱项识别与改进建议
- 成绩曲线图生成与导出

## 安装与运行

### 开发环境运行
```bash
pip install -r requirements.txt
python main.py
```

### 打包发布
```bash
pyinstaller --windowed --onefile --name "体育成绩评估系统" main.py
```

## 系统要求
- Python 3.7+
- macOS 10.14+ / Windows 10+
