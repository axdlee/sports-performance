# 体育成绩评估系统 - 使用说明

## 系统简介

体育成绩评估系统是一款专为中学/高校设计的桌面端体育成绩管理软件，支持macOS和Windows平台。系统根据《国家学生体质健康标准》自动计算体育成绩，提供成绩分析、历史记录管理和弱项改进建议。

## 主要功能

### 1. 用户管理
- 支持多用户注册和登录
- 用户信息管理（姓名、学号、性别）
- 历史成绩记录管理

### 2. 成绩录入
- **必选项**：男生1000米跑 / 女生800米跑（10分）
- **第一类选考**（10分）：50米跑、坐位体前屈、立定跳远、引体向上（男）/仰卧起坐（女）
- **第二类选考**（10分）：篮球运球、足球运球、排球垫球
- 实时成绩计算和得分显示
- 总分自动计算（满分30分）

### 3. 成绩分析
- 等级评定（优秀≥27、良好24-26.5、中等18-23.5、及格15-17.5、不及格<15）
- 弱项识别和改进建议
- 历史成绩趋势分析
- 成绩分布图表生成

### 4. 数据管理
- JSON格式本地数据存储
- 历史记录查询和统计
- 数据导入导出功能

## 安装与运行

### 方法一：直接运行（推荐）

1. **安装Python环境**
   ```bash
   # macOS
   brew install python3
   
   # Windows
   # 从 https://python.org 下载安装
   ```

2. **安装依赖包**
   ```bash
   pip install matplotlib pillow pyinstaller
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 方法二：使用虚拟环境

1. **创建虚拟环境**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # macOS/Linux
   # 或
   venv\Scripts\activate     # Windows
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 方法三：安装包使用

1. **生成安装包**
   ```bash
   python build.py
   ```

2. **使用安装包**
   - macOS: 双击 `体育成绩评估系统.app` 即可运行
   - Windows: 双击 `体育成绩评估系统.exe` 即可运行
   - 无需安装Python环境，开箱即用

## 使用指南

### 1. 首次使用

1. 启动程序后，点击"用户登录"
2. 输入姓名、选择性别
3. 点击"注册新用户"创建账户

### 2. 录入成绩

1. 登录后点击"成绩录入"
2. 输入必选项成绩（1000米/800米跑）
3. 选择第一类选考项目并输入成绩
4. 选择第二类选考项目并输入成绩
5. 系统自动计算各项得分和总分
6. 点击"保存成绩"完成录入

### 3. 查看报告

1. 点击"成绩报告"查看详细分析
2. **当前成绩**标签页：查看最新成绩和等级评定
3. **历史趋势**标签页：查看历史记录和生成趋势图
4. **弱项分析**标签页：查看弱项识别和改进建议

## 成绩录入格式说明

### 时间格式
- `3'45"` - 3分45秒
- `3:45` - 3分45秒  
- `225` - 225秒（纯秒数）

### 距离格式
- `250` - 250厘米
- `2.5` - 2.5厘米

### 次数格式
- `15` - 15次
- `0` - 0次

## 评分标准

### 等级评定
- **优秀**：27-30分
- **良好**：24-26.5分
- **中等**：18-23.5分
- **及格**：15-17.5分
- **不及格**：0-14.5分

### 项目权重
- 必选项：10分
- 第一类选考：10分
- 第二类选考：10分
- **总分**：30分

## 数据存储

- 数据文件位置：`data/users.json`
- 图表文件位置：`data/` 目录
- 支持多用户数据存储
- 自动备份和恢复

## 常见问题

### Q: 程序无法启动？
A: 检查Python环境是否正确安装，确保已安装所有依赖包。

### Q: 图表无法生成？
A: 确保已安装matplotlib库，检查数据目录权限。

### Q: 成绩计算不准确？
A: 检查输入格式是否正确，参考评分标准表。

### Q: 数据丢失？
A: 检查`data/users.json`文件是否存在，程序会自动创建数据文件。

## 技术支持

如遇到问题，请检查：
1. Python版本（建议3.7+）
2. 依赖包安装情况
3. 数据文件完整性
4. 系统权限设置
