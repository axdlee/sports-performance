# 杀毒软件误报说明

## ⚠️ 重要提示

本程序是使用 Python 和 PyInstaller 打包的合法应用程序，**不包含任何病毒或恶意代码**。

## 🛡️ 为什么会被杀毒软件拦截？

PyInstaller 打包的程序经常被杀毒软件误报，主要原因：

1. **动态解压技术**：程序运行时会解压内部文件到临时目录
2. **缺少数字签名**：个人开发者难以获得昂贵的代码签名证书
3. **新生成的文件**：杀毒软件对新的、不常见的 exe 文件更加警惕
4. **PyInstaller 特征**：许多恶意软件也使用 PyInstaller 打包，导致误报

**这是 PyInstaller 打包程序的常见问题，不代表程序有问题。**

## ✅ 如何安全运行程序

### Windows Defender

#### 方法 1：允许运行（快速）

1. 当 Windows Defender 弹出警告时
2. 点击 **"更多信息"** 或 **"详细信息"**
3. 点击 **"仍要运行"** 按钮

#### 方法 2：从隔离区恢复

如果程序已被删除：

1. 打开 **Windows 安全中心**
2. 进入 **病毒和威胁防护**
3. 点击 **保护历史记录**
4. 找到被隔离的文件
5. 点击 **还原** 或 **允许**

#### 方法 3：添加排除项（推荐）

永久解决问题：

1. 打开 **Windows 安全中心**
2. 进入 **病毒和威胁防护** → **管理设置**
3. 滚动到 **排除项**，点击 **添加或删除排除项**
4. 点击 **添加排除项** → 选择 **文件夹**
5. 选择程序安装目录（例如：`C:\Program Files\体育成绩评估系统`）
6. 确认添加

### 其他杀毒软件

**360 安全卫士 / 腾讯电脑管家：**
1. 在拦截提示中选择 **"信任此程序"** 或 **"加入白名单"**
2. 或在软件设置中添加信任区

**火绒安全软件：**
1. 右键点击被拦截的文件
2. 选择 **"信任文件"**
3. 或在设置 → 信任区中添加

**卡巴斯基 / Norton / McAfee：**
1. 在隔离区中找到文件
2. 选择恢复并添加到例外/信任列表

## 🔒 安全验证

如果你仍然担心安全性，可以：

### 1. 查看源代码

本项目完全开源，可以在 GitHub 上查看所有源代码：
- 仓库地址：https://github.com/axdlee/sports-performance
- 所有代码都是透明可审查的

### 2. 自行打包

如果你有 Python 环境，可以自己从源代码打包：

```bash
# 克隆仓库
git clone https://github.com/axdlee/sports-performance.git
cd sports-performance

# 安装依赖
pip install -r requirements.txt

# 打包
python build.py
```

### 3. 使用在线扫描工具

上传 exe 到以下在线扫描服务（多引擎扫描）：
- VirusTotal: https://www.virustotal.com/
- Jotti: https://virusscan.jotti.org/
- MetaDefender: https://metadefender.opswat.com/

**注意**：即使在这些平台上，也可能有少数引擎误报，这是正常现象。

## 📊 检测结果说明

如果使用 VirusTotal 扫描，常见结果：
- ✅ **0-5 个引擎报告**：完全正常，大多是误报
- ⚠️ **5-10 个引擎报告**：可能是启发式误报
- ❌ **10+ 个主流引擎报告**：可能存在问题（但本程序不会出现这种情况）

典型的 PyInstaller 误报引擎：
- Windows Defender
- Avira
- Rising
- Jiangmin
- 部分国内杀毒软件

## 🔐 未来改进计划

为了减少误报，我们计划：

1. **代码签名**：申请代码签名证书（需要成本）
2. **提交样本**：向主流杀毒厂商提交程序样本，申请白名单
3. **优化打包**：持续优化 PyInstaller 配置，减少可疑特征

## 💡 为什么不使用其他打包方式？

- **Nuitka**：编译时间长，可能有兼容性问题
- **py2exe**：只支持 Windows，跨平台性差
- **cx_Freeze**：类似的误报问题
- **PyInstaller**：目前最成熟稳定的方案，误报是行业共性问题
